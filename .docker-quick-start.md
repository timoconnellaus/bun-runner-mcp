# Docker Quick Start

Quick reference for common Docker sandbox operations.

## Essential Commands

```bash
# Start the sandbox
make up

# Stop the sandbox
make down

# View logs
make logs

# Run test code
make test

# Rebuild and restart
make dev-rebuild
```

## Using the Helper Script

```bash
# Quick test
./scripts/run-sandbox.sh "console.log('Hello!')"

# From file
./scripts/run-sandbox.sh -f my-script.ts

# Start in background
./scripts/run-sandbox.sh -d

# Stop
./scripts/run-sandbox.sh -s
```

## Direct API Access

```bash
# Execute code
curl -X POST http://localhost:9998/execute \
  -H "Content-Type: application/json" \
  -d '{"code": "console.log(Bun.version)"}'

# Health check
curl http://localhost:9999/health
```

## Troubleshooting

```bash
# Check status
make ps

# View detailed logs
make logs-tail

# Check health
make health

# View resource usage
make stats

# Inspect configuration
make inspect

# Open shell in container
make shell
```

## Security Checks

```bash
# Run all security checks
make security-check

# Scan for vulnerabilities
make scan

# View security settings
docker inspect bun-sandbox | jq '.[0].HostConfig'
```

## Cleanup

```bash
# Stop and remove container
make down

# Remove everything including images
make clean

# Remove unused Docker resources
make prune
```
